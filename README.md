# 📖 OOP school library <a name="about-project"></a>

**OOP school library** is an app that keeps records of what books are in the library and who borrows them. It also allows one to add students\teachers, add books and save records of who borrowed a book.

### Key Features <a name="key-features"></a>

- **Add new students or teachers.**
- **Add new books.**
- **Save records of who borrowed a given book and when.**
